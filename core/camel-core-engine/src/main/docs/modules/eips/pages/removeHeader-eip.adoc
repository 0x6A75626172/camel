= Remove Header EIP
:doctitle: Remove Header
:shortname: removeHeader
:description: Removes a named header from the message
:since: 
:supportlevel: Stable

The Remove Header EIP allows you to remove a header from the xref:message.adoc[Message].

== Options

// eip options: START
include::partial$eip-options.adoc[]
// eip options: END

== Example

We want to remove a header with key "myHeader" from the message:

[source,java]
----
from("seda:b")
  .removeHeader("myHeader")
  .to("mock:result");
----

And in XML:

[source,xml]
----
<route>
  <from uri="seda:b"/>
  <removeHeader name="myHeader"/>
  <to uri="mock:result"/>
</route>
----
